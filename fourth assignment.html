Differences and similarities between ==,===,!=,!==,

==(Equality operator): the == operator compares the value of two operands for equality after converting both operands to a comon type.
===(strict equality operator): the === operator also compares the value of two operands for equality,but without type coercion.
!= (inequality operator):the != operator is the opposite of the == operator.it return true if the operands are not equal,after type coercion.
!==(strict inequality operator): the !== operator is the opposite of the === operator. it returns true if the operands are not equal,without type coercion.
