1. 10 reserved javascript keywords and what they do 1. 'var' - used for declaring variables in Javaascript.
2. 'let' - used for declaring variable,introduced in ES6,with blocklevel scoping.
3. 'const'- used for declaring constant. introduced in ES6.
4. 'function'- used for declaring a function in javascript.
5. 'if'- used to execute a statement if a certain condition is true,or to execute a differnt statement if the condition is false.
6. 'else'- used to execute a statement if the 'if' condition is false.
7. 'switch'- used to execute one of several possible statement depending on the value of an expression.
8. 'return'- used to return a value rom a function or to exit a function
9. 'case'- used with 'switch' to specify a match for the expression
10. 'default'- used with 'switch' to specify a statement to execute if no match is found.
