3. let let is used to declare a block-scoped variable.which can be reassignedto a new value within the same block.Here's an example:
let x=10; if(true){ let x =20 console.log(x):// Output:20 } console.log(x):// Output:10
in this example x is a variable declared with let thatbhas a block scope. The console log(x) inside the if statement outputs 20 because it is referencing the x declared within the same block. The console.log(x) outside the if statement outputs 10 because it is referencing the x declared outside the block.
